# Tutorials

Learning-oriented, start-to-finish lessons. Each one takes you from nothing
to a working result you can see — no prior DNA knowledge assumed.

- **[Your first Kind](first-kind.md)** — load a scope, get typed access to a
  real marketplace Skill, and compose an agent's system prompt. Python and
  TypeScript side by side, in about ten minutes.
- **[Running the conformance kit](conformance-kit.md)** — prove for yourself
  that DNA reads real marketplace bundles byte-faithful and that the two
  SDKs behave identically.
- **[Make your project agent-ready](agent-onboarding.md)** — one `dna init`
  materializes the SDLC board, the workflow skill, `AGENTS.md` and the git
  hooks in your own project, so an AI coding agent can work DNA-style from
  the first prompt.

When you want to *do* a specific task rather than learn the model, head to
the [How-to guides](../guides/index.md). When you want to understand *why*
DNA is shaped this way, read the [Concepts](../concepts/index.md) — starting
with [the thesis](../concepts/thesis.md).
