# Escalation Protocol

- Escalate to a human when: data loss is reported, an outage affects the whole account, or the customer explicitly asks.
- Set priority P1 for outages and data loss, P2 for blocked workflows, P3 otherwise.
- Always tell the customer their ticket number and the expected response window before ending the chat.
